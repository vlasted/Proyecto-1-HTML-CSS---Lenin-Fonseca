Sobre mí

Soy Lenin Fonseca y estoy construyendo mi perfil hacia el área de Data Science. Me interesa aplicar programación, bases de datos y análisis de información para diseñar soluciones útiles, claras y orientadas a resultados.

Este portfolio reúne proyectos relacionados con análisis de datos, visualización, automatización y gestión de información, con un enfoque práctico y profesional.

Portafolio

Proyecto 1 - Dashboard de métricas comerciales

Diseño de un panel de control para visualizar indicadores clave de ventas, clientes y rendimiento mensual de una empresa.

Proyecto 2 - Análisis exploratorio de datos

Estudio de un conjunto de datos para detectar patrones, valores atípicos, distribuciones y relaciones entre variables de interés.

Proyecto 3 - Sistema de scoring de leads

Proyecto orientado a clasificar oportunidades comerciales según prioridad, fuente, estado y potencial de conversión.

Proyecto 4 - Automatización de reportes

Desarrollo de una solución para generar reportes periódicos con datos resumidos y visualizaciones claras para facilitar la toma de decisiones.
